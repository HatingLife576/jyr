Jyr is a system of software working together to better Linux. Started by 5LT these software will make people work together easier. One project in Jyr is UPMFL (Universal Package Manager For Linux) where .upmfl files will be used on (hopefully) most distros. This will make making software for other distrobutions is easier. 5LT is also makeing her own distros. Sadly if you known 5LT for a while you would know about the Vyx project which is fully shutting down May 05, 2019. 0.0.5 will not be released.
